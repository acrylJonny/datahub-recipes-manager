About GitHub Integration

Integrating with GitHub allows you to:

  • Version control your DataHub recipes
  • Track changes through pull requests
  • Collaborate with team members on recipe changes
  • Automate deployment with GitHub Actions
The personal access token is stored securely and is only used to interact with your GitHub repository.
Creating a Token
  1. Go to GitHub Settings > Developer settings > Personal access tokens
  2. Click "Generate new token"
  3. Give it a descriptive name (e.g., "DataHub Recipes Manager")
  4. Select at least the repo scope
  5. Click "Generate token" and copy the token
